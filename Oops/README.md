# C++ OOP Lab Programs

This repository contains a collection of basic **C++ Object-Oriented Programming (OOP)** lab experiments.
These programs are designed to demonstrate core OOP concepts with simple and clear examples.

---

## Topics Covered

* Classes & Objects
* Constructors (Default, Parameterized, Copy)
* Destructor
* Friend Functions
* Multiple Inheritance
* Virtual Functions
* Dynamic Memory Allocation
* Operator Overloading
* Exception Handling

---

## Program List

### 🔹 Program 1 – Class (Room)

* Calculates **area and volume** of a room using class and objects.

### 🔹 Program 2 – Rectangle

* Uses **constructor** to initialize values and calculate area.

### 🔹 Program 3 – Friend Function

* Demonstrates how a **friend function** can access private data.

### 🔹 Program 4 – Multiple Inheritance

* Shows how a class can inherit from **multiple base classes**.

### 🔹 Program 5 – Virtual Function

* Demonstrates **runtime polymorphism** using virtual functions.

### 🔹 Program 6 – Dynamic Memory Allocation

* Uses `new` and `delete` for **memory management**.

### 🔹 Program 7 – Copy Constructor

* Demonstrates copying one object to another.

### 🔹 Program 8 – Destructor

* Shows how destructors are called automatically.

### 🔹 Program 9 – Operator Overloading

* Overloads `+` operator for **complex number addition**.

### 🔹 Program 10 – Exception Handling

* Handles **division by zero** using try-catch.

---

## How to Run

1. Install a C++ compiler (GCC / G++)
2. Compile any program:

   ```bash
   g++ program1.cpp -o program1
   ```
3. Run:

   ```bash
   ./program1
   ```
